# SCT_DS_01
# Task 01 – Data Visualization

## Objective
Create a bar chart or histogram to visualize the distribution of a categorical or continuous variable, such as the distribution of ages or genders in a population
## Tools Used
- Power BI
- Population dataset (World Bank)

## Description
This project visualizes population distribution using a bar chart in Power BI.  
The dataset contains population values for different countries.

## Files Included
- population_visualization.pbix – Power BI project file
- population_dataset.csv – dataset used
- chart_output.png – visualization output

## Result
The visualization shows the top 10 countries with the highest population.
